Administrator Workflow Documentation

BetSure Call Center Management System

Version 1.0 | Last Updated: October 2025

1. Overview

As an Administrator, you have complete system access and control over all aspects of the BetSure Call Center platform. Your role is critical in maintaining system integrity, managing users, and ensuring smooth operations across all departments.

2. Core Responsibilities

  • User Management: Create, approve, and manage all user accounts (Admins, Managers, Agents)
  • System Configuration: Configure system-wide settings and integrations
  • Campaign Oversight: Monitor and manage all active campaigns
  • Lead Distribution: Oversee lead assignment and distribution processes
  • Reporting: Generate and analyze comprehensive system reports
  • Security: Maintain system security and access controls
  • Performance Monitoring: Track overall system performance and user activity

3. Daily Workflow

Step 1: System Login

  1. Navigate to the BetSure Call Center portal
  2. Enter admin credentials
  3. Verify successful authentication
  4. Review system dashboard for alerts

Step 2: Review Pending Approvals

  1. Navigate to User Management section
  2. Review pending user registrations
  3. Verify user information and role assignments
  4. Approve or reject user accounts
  5. Assign appropriate roles (Admin/Management/Agent)

Step 3: Monitor Active Agents

  1. Access the Dashboard
  2. View real-time agent status (Online, On-call, Offline)
  3. Track active call durations
  4. Monitor agent performance metrics
  5. Identify any agents requiring support

Step 4: Lead Management

  1. Import new lead lists via CSV upload
  2. Review lead distribution across agents
  3. Monitor lead assignment balance
  4. Reassign leads if necessary
  5. Verify campaign associations

Step 5: Campaign Management

  1. Review active campaigns
  2. Create new campaigns with:
    • Campaign name and description
    • Target segments and goals
    • AI-generated scripts
    • Start and end dates
  3. Monitor campaign performance metrics
  4. Adjust campaign parameters as needed

Step 6: Generate Reports

  1. Navigate to Reports section
  2. Select report type and date range
  3. Choose data inclusions (Metrics, Agents, Queues, Call Notes)
  4. Generate AI-powered analytical reports
  5. Review insights and recommendations
  6. Export reports for stakeholders

Step 7: System Configuration

  1. Access Settings panel
  2. Configure:
    • Call routing rules
    • Lead distribution algorithms
    • Performance thresholds
    • Notification preferences
    • Integration settings
  3. Save and apply changes

4. User Management Procedures

4.1 Creating New Users

  1. Navigate to User Management > Add User
  2. Enter user details:
    • Full name
    • Email address
    • Role (Admin/Management/Agent)
  3. Assign initial password
  4. Set approval status
  5. Save user profile

4.2 Managing User Roles

Role Permissions Access Level
Admin Full system access, user management, all reports Unrestricted
Management Monitor agents, view all reports, manage campaigns Supervisory
Agent View assigned leads, make calls, update records Limited
Important: Always verify user identity before approving accounts. Incorrect role assignments can compromise system security.

5. Lead Distribution Management

5.1 Importing Leads

  1. Prepare CSV file with required fields:
    • Name, Phone, Segment, Priority, Campaign
  2. Navigate to Leads > Import
  3. Upload CSV file
  4. Map columns to system fields
  5. Preview import data
  6. Confirm import
  7. System automatically distributes leads to available agents

5.2 Manual Lead Assignment

  1. Access Leads list
  2. Select unassigned leads
  3. Choose target agent(s)
  4. Assign leads
  5. Verify assignment in agent dashboard

5.3 Lead Distribution Rules

  • Leads distributed equally among approved agents
  • Campaign-specific assignments honored
  • Priority leads assigned first
  • SLA deadlines considered in assignment

6. Campaign Management

6.1 Creating Campaigns

  1. Navigate to Campaigns > Create New
  2. Enter campaign details:
    • Campaign name
    • Description
    • Target segment
    • Start and end dates
    • Target metrics (calls, conversions)
  3. Generate AI script (optional)
  4. Review and edit script
  5. Activate campaign
  6. Assign leads to campaign

6.2 Monitoring Campaign Performance

  • Track total calls vs. target
  • Monitor conversion rates
  • Review deposit amounts
  • Analyze agent performance per campaign
  • Identify trends and patterns

7. Reporting and Analytics

7.1 Available Reports

Performance Reports

Agent performance metrics, call statistics, conversion rates

Campaign Reports

Campaign-specific analytics and ROI metrics

Agent Activity Reports

Individual agent performance, call logs, notes summary

System Health Reports

System uptime, error logs, integration status

7.2 Generating AI-Powered Reports

  1. Select report date range
  2. Choose data to include:
    • ✓ Include Metrics (call stats, conversions)
    • ✓ Include Agents (individual performance)
    • ✓ Include Queues (campaign data)
    • ✓ Include Call Notes (AI-summarized insights)
  3. Generate report
  4. AI analyzes data and provides:
    • Performance insights
    • Trend analysis
    • Recommendations
    • Call notes summary
  5. Export as HTML or PDF

8. System Settings and Configuration

8.1 Integration Settings

  • Africa's Talking SIP: Configure VoIP credentials
  • OpenAI API: Manage AI script generation settings
  • Supabase: Database and authentication configuration

8.2 Notification Settings

  • Configure system alerts
  • Set notification thresholds
  • Manage email notifications
  • Define escalation rules

8.3 Security Settings

  • Manage user roles and permissions
  • Configure password policies
  • Review access logs
  • Enable two-factor authentication

9. Monitoring and Troubleshooting

9.1 Real-Time Monitoring

  • Agent status dashboard (Online/On-call/Offline)
  • Active call monitoring
  • Call duration tracking
  • Queue metrics
  • System performance indicators

9.2 Common Issues and Solutions

Issue Solution
Agent can't access system Check approval status in User Management
Leads not distributing Verify approved agents are available
Call quality issues Check SIP credentials and network connectivity
Reports not generating Verify date range and data availability

10. Best Practices

  • ✓ Review pending approvals daily
  • ✓ Monitor system health continuously
  • ✓ Generate weekly performance reports
  • ✓ Maintain balanced lead distribution
  • ✓ Regularly review campaign performance
  • ✓ Keep user roles and permissions up to date
  • ✓ Document system changes and configurations
  • ✓ Provide timely support to managers and agents
  • ✓ Backup critical data regularly
  • ✓ Stay informed about system updates

Critical Reminders

  • ⚠️ Only approve users after verifying their identity
  • ⚠️ Review and test campaign scripts before activation
  • ⚠️ Monitor lead distribution for fairness
  • ⚠️ Backup system configuration before making changes
