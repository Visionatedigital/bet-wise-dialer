1. Overview

As a Manager, you are responsible for overseeing agent performance, monitoring call quality, managing campaigns, and ensuring team targets are met. You have comprehensive access to all agent activities, real-time monitoring capabilities, and advanced reporting tools.

Your role bridges the operational execution of agents with the strategic oversight of administrators.

2. Core Responsibilities

  • Agent Monitoring: Track real-time agent status, call activity, and performance metrics
  • Performance Management: Review agent KPIs and provide coaching based on AI-analyzed insights
  • Campaign Management: Create, monitor, and optimize campaigns
  • Quality Assurance: Review call notes and ensure compliance standards
  • Reporting: Generate and analyze AI-powered reports with call note summaries
  • Team Development: Identify training needs and improvement opportunities
  • Target Setting: Monitor team progress toward daily/weekly/monthly goals

4. Real-Time Agent Monitoring

4.1 Understanding Agent Status

Status Meaning What to Monitor
üü¢ Online Agent is logged in and available to make calls Ensure agents start calling promptly, not idle
üîµ On-call Agent is actively on a call with a customer Watch call duration; intervene if too long (>15 min)
‚ö™ Offline Agent is logged out or unavailable Verify offline status matches scheduled breaks/shift end

4.2 Key Monitoring Metrics

Current Call Duration:
Displayed in real-time for agents on active calls. Alert if exceeds 15 minutes - agent may need supervisor assistance.
Last Status Change:
Shows when agent went online or started current call. Helps track actual working time vs. scheduled time.
Calls Today:
Running count of completed calls. Compare to targets and identify underperformers early in the shift.
Assigned Leads:
Number of leads in agent's queue. Ensure fair distribution and adequate pipeline.

4.3 When to Intervene

  • Call Duration >15 minutes: Check if agent needs help closing or handling objection
  • Low Call Volume: If agent has made <50% of target by midday, investigate
  • Offline During Shift: Verify legitimate break vs. unauthorized absence
  • No Leads Assigned: Check system for distribution issues

5. Performance Analysis and Coaching

5.1 Key Performance Indicators (KPIs)

Calls Made

Target: 80-100 per agent/day

Low? Check if agent has enough leads or technical issues

Connect Rate

Target: 50%+

Low? Review calling times and lead quality

Conversion Rate

Target: 20%+

Low? Listen to calls, review script adherence, provide coaching

Avg Handle Time

Target: 5-7 minutes

Too long? Coach on efficient closing. Too short? Check if skipping steps

Total Revenue

Target: UGX 500k+ per agent/day

Track deposit amounts and conversion quality

Compliance Score

Target: 100%

Review call notes; ensure all checklist items covered

5.2 Using Call Notes for Coaching

Agent call notes provide invaluable coaching insights:

  • Objection Patterns: What objections are agents hearing most? Develop targeted responses
  • Note Quality: Are notes detailed enough? Coach agents on proper documentation
  • Follow-up Discipline: Are agents scheduling callbacks properly?
  • Customer Sentiment: Look for patterns in customer reactions
  • Success Stories: Identify what's working and share with team
Pro Tip: When generating reports, always include call notes. The AI summarizes common themes, objections, and success patterns across hundreds of calls - providing coaching insights you'd miss by reviewing calls individually.

5.3 Coaching Best Practices

  1. Use Data: Base coaching on actual metrics and call notes, not assumptions
  2. Be Specific: "Your handle time is 12 minutes vs. target of 7. Let's work on efficient closing."
  3. Positive Reinforcement: Highlight what agents are doing well
  4. Action Plans: Set specific improvement goals with timelines
  5. Follow Up: Monitor progress and provide ongoing support

6.2 How AI Analyzes Call Notes

When you include call notes in reports, the AI:

  1. Aggregates: Collects all call notes from selected date range
  2. Categorizes: Groups notes by themes (objections, interests, commitments)
  3. Identifies Patterns: Finds common customer responses and concerns
  4. Sentiment Analysis: Assesses overall customer sentiment trends
  5. Generates Insights: Provides actionable recommendations
  6. Summarizes: Creates concise executive summary of findings

7. Campaign Management

7.1 Creating Effective Campaigns

Step-by-Step Campaign Creation:
  1. Navigate to Campaigns > Create New
  2. Define campaign basics:
    • Name (descriptive, e.g., "Football Weekend Bonus Push")
    • Description (clear goals and context)
    • Target segment (Hot/Warm/Cold leads)
  3. Set measurable targets:
    • Total calls target
    • Conversion target
    • Revenue target
  4. Configure AI script:
    • Click "Generate AI Script"
    • Review and customize
    • Test with pilot agents
  5. Set campaign duration (start/end dates)
  6. Activate and assign leads

7.2 Monitoring Campaign Performance

  • Track progress daily against targets

  • Review agent feedback on script effectiveness
